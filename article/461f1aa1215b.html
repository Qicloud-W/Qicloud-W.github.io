获取QQ昵称（php笔记）

发表于2021-04-27|更新于2024-05-22|絮絮叨叨
|阅读量:
function get_qq_nick($uid)
{
        $get_info = file_get_contents('https://r.qzone.qq.com/fcg-bin/cgi_get_portrait.fcg?uins='.$uid);
        $nic = explode(',',$get_info);
        return trim(mb_convert_encoding($nic[6], "UTF-8", "GBK"),'"');
}

写点小玩意，自动取QQ昵称这玩意贼好使 别问为啥不用curl 因为懒，但是我自己确实用的curl 哈哈哈

Content-Type

form-data

地址 协议 方法 线路
https://blog.zets.cn/api/qqinfo https GET/POST 国内

请求Query参数

参数名 示例值 参数类型 是否必填 参数描述
qq 66547997 string 是 qq号
size true string 否 默认100px，若有值图像大小为640px

成功响应示例

{
    "code": 200,
    "msg": "susses",
    "data": {
        "uid": "66547997",
        "nick": "Qicloud",
        "avatar": "https://q4.qlogo.cn/g?b=sdk&k=UeVu6eVo5SdL2Eu7PiczvTw&s=640"
    }
}
参数名 示例值 参数类型 参数描述
code 200 string 状态值
msg susses string 返回文字描述
data object 返回数据
data.uid 66547997 number qq
data.nick Qicloud string 昵称
data.avatar https://q4.qlogo.cn/g?b=sdk&k=UeVu6eVo5SdL2Eu7PiczvTw&s=640 string 加密后的外链头像
